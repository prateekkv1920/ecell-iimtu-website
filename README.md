# E-Cell IIMTU — Official Website

[![GitHub Pages](https://img.shields.io/badge/Live-GitHub%20Pages-brightgreen)](https://prateekkv1920.github.io/ecell-iimtu/)

The flagship website for the **Entrepreneurship Cell of IIMT University, Meerut**.

## Features

- 🏠 **Homepage** — Hero, stats, featured event, newsletter signup
- 🗓️ **Events** — Upcoming/past events with filters, search, countdown timers & QR registration
- 🚀 **Startups** — Startup showcase with voting, domain filter, and idea submission form
- 👥 **Team** — Faculty mentors with 1:1 booking modal, student body
- 📝 **Blog** — Articles with category filter and in-modal article reader
- 📬 **Contact** — Join E-Cell application form with FAQ accordion
- 📧 **Newsletter Archive** — Past newsletters readable in browser
- 🔒 **Admin Panel** — Full CMS: event manager, newsletter composer, blog editor, registrations, bookings

## Tech Stack

- **HTML5 + Vanilla CSS + JavaScript** (no build tools required)
- **Google Fonts** (Syne + DM Sans)
- **Canvas API** for QR code generation
- **localStorage** for voting, session management

## Quick Start

Open `index.html` in any browser, or deploy to GitHub Pages / Netlify / Vercel.
